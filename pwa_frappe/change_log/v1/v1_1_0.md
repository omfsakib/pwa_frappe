# Version 1.1.0 Release Notes

### Features & Enhancements

- Firebase Cloud Messaging (FCM) push notifications for desk `Notification Log` entries (mentions, assignments, shares, energy points, alerts) — subscribes each logged-in desk user automatically, no manual opt-in step
- New **FCM User Token** doctype to store per-user, per-device push tokens; stale/unregistered tokens are pruned automatically
- New **Firebase Cloud Messaging (FCM)** section on the **Service Worker** doctype for VAPID key, Firebase web config, and Firebase service account credentials (stored encrypted)
- New `pwa_frappe.fcm` module: `subscribe`, `unsubscribe`, `get_fcm_config`, and a `test_push_notification` helper for manual testing via `bench execute`
- Background push notifications handled in `sw.js` via Firebase Messaging, including click-to-open support
- New 300x300 SVG app logo (`public/images/logo.svg`)

### Documentation

- README rewritten with a full FCM push notification setup guide (Firebase project setup, Service Worker configuration, end-user experience) with screenshots
